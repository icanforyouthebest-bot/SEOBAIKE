## 安裝與使用

```bash
# 複製專案
git clone https://github.com/icanforyouthebest-bot/SEOBAIKE.git

# 執行十層任務鏈
cd SEOBAIKE
python task_generator.py      # 第一層
python assign_tasks.py        # 第二層
python monitor_tasks.py       # 第三層
python optimize_scheduler.py  # 第四層
python clone_checker.py       # 第七層
python self_audit.py          # 第九層
```
