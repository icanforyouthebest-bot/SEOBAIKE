## 二、`is_available` 欄位含義

- `true`：此平台記錄已收錄至目錄，資料來源為公開搜索
- `false`：目前不適用（所有記錄預設為 true）

此欄位**不代表**：
- 該平台是否仍在營運
- 我們是否有該平台的帳號
- API 是否可用

## 三、收錄來源

所有 4,000 筆記錄透過以下方式產生：

1. **WebSearch 搜索**：使用搜索引擎搜索各產業 AI/SaaS 平台
2. **公開資料整理**：從搜索結果中擷取公司名、產品名、功能
3. **SQL INSERT**：透過 Supabase migration 批次匯入
