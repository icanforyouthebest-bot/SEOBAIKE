## 一、四層治理模型（L1-L4）

```
L4 — 治理與自動化層
      |（策略 -> 流程）
      v
L3 — 流程與執行層
      |（事件 -> 外部）
      v
L2 — 推論與規則層
      |（唯讀 -> 推論）
      v
L1 — 主權資料層
```

---

## 二、資料流（Data Flow）

```
L1 -> L2 -> L3 -> External
```

單向。
不可逆。
不可污染。

---

## 三、禁止路徑（Prohibited Paths）

```
L4 --X--> L1
L4 --X--> L2
L3 --X--> L1（修改結構）
L2 --X--> L1（寫入）
L2 --X--> L3（流程控制）
L1 --X--> 上層引用
```

越權即污染。
跳層即失控。
混層即崩壞。
