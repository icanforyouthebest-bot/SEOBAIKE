# MCP 主權 OS

## RISK MATRIX

**風控矩陣（越權 / 跳層 / 污染）**

---

## 一、風險宇宙（Risk Universe）

所有風險都來自三件事：

越權（Overreach）
跳層（Layer Skipping）
污染（Contamination）

沒有第四種。

---

## 二、風險矩陣總覽（Risk Matrix Overview）

|  | L1（資料） | L2（推論） | L3（流程） | L4（治理） |
|--|-----------|-----------|-----------|-----------|
| 越權 Overreach | 最高風險 | 高風險 | 中風險 | 中風險 |
| 跳層 Skipping | 最高風險 | 高風險 | 高風險 | 中風險 |
| 污染 Contam. | 致命風險 | 高風險 | 中風險 | 中風險 |

L1 污染 = 系統死亡。
L2 越權 = 治理崩壞。
L3 跳層 = 流程失控。
L4 越權 = 主權破壞。

---

## 三、風險 1：越權（Overreach）

越權 = 權力跑錯層。
權力跑錯層 = 風險。

```
L2 -> L1（寫入） X
L3 -> L1（修改結構） X
L4 -> L1（直接操作） X
L4 -> L2（干預推論） X
L4 -> L3（跳過流程） X
```

越權 = 攻擊面。
攻擊面 = 漏洞。

---

## 四、風險 2：跳層（Layer Skipping）

跳層 = 破壞秩序。
破壞秩序 = 失控。

```
流程（L3） -> 跳過推論（L2） X
推論（L2） -> 跳過資料（L1） X
治理（L4） -> 跳過流程（L3） X
```

跳層 = 系統無法審計。
無法審計 = 無法治理。

---

## 五、風險 3：污染（Contamination）

污染 = 死亡。
這是 OS 最大的敵人。

```
資料（L1） <- 邏輯污染 X
推論（L2） <- 流程污染 X
流程（L3） <- 治理污染 X
```

污染一旦發生，
整個 OS 必須重建。

---

## 六、風險等級（Risk Severity Levels）

**Level 5 — 致命（Fatal）**

- L1 資料污染
- L1 越權寫入
- L3 跳過 L2
- L4 直接操作 L1

**Level 4 — 高風險（Critical）**

- 推論污染
- 流程污染
- 治理污染
- 黑箱邏輯

**Level 3 — 中風險（Moderate）**

- 模糊邊界
- 模糊責任
- 模糊流程

**Level 2 — 可控（Low）**

- 可審計
- 可驗證
- 可替換

**Level 1 — 安全（Safe）**

- 四層分離
- 單向資料流
- 禁止路徑
- 語義治理

## 七、風控三問（Risk Questions）

每一個模組、每一條流程、每一段邏輯
都必須回答：

**Q1：它會越權嗎？**

會 -> 封鎖。

**Q2：它會跳層嗎？**

會 -> 移除。

**Q3：它會污染嗎？**

會 -> 重建。
