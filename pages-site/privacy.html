Section 1

一、前言與資料收集範圍

歡迎使用 SEOBAIKE 平台（以下簡稱「本平台」）。本平台由小路光有限公司（統一編號：60475510，以下簡稱「本公司」）營運。本公司致力於保護您的個人資料及隱私權。

本隱私權政策說明當您造訪 aiforseo.vip 網站及使用本平台服務時，本公司如何收集、處理、利用、儲存及保護您的個人資料。本政策適用於全球所有使用者。

核心承諾：本公司僅收集提供服務所必要之最少量資料。本公司不出售您的個人資料予任何第三方。

1.1 您主動提供之資料

  • 帳號註冊資料：電子信箱、使用者名稱、密碼（經加密儲存）
  • 個人檔案資料：顯示名稱、國家/地區（選填）、語言偏好
  • 聯絡資訊：姓名、公司名稱（僅於您主動聯繫時收集）
  • 市集資料：賣家之商品描述、服務內容、定價資訊
  • API 使用資料：API 金鑰、呼叫記錄

1.2 自動收集之資料

  • 頁面瀏覽記錄：造訪頁面路徑、來源網址、瀏覽器資訊
  • 裝置資訊：作業系統類型、螢幕解析度、瀏覽器類型及版本
  • 網路資訊：IP 位址（經處理後僅保留國家/城市層級）
  • 使用行為：功能使用頻率、API 呼叫次數、錯誤記錄

1.3 我們不收集的資料

  • 信用卡號碼或銀行帳號：所有金融支付資訊由第三方金流服務商處理
  • 身分證字號或護照號碼：本公司不要求亦不收集政府核發之身分證明文件
  • 生物辨識資料：不收集指紋、臉部辨識或其他生物辨識資訊
  • 健康或醫療資料：不收集任何健康或醫療相關資訊

Section 3

三、資料保護措施

3.1 資料儲存位置

您的帳號資料、交易記錄及平台使用資料儲存於本公司之雲端資料庫，伺服器位於日本東京。本平台採用企業級關聯式資料庫，提供列級安全策略 (Row Level Security, RLS) 保護。

3.2 安全措施

  • 傳輸加密：所有網路通訊均透過 TLS/SSL 加密傳輸
  • 密碼保護：密碼經 bcrypt 雜湊處理，不以明文儲存
  • API 金鑰安全：API 金鑰採用安全亂數生成，並支援金鑰輪換
  • 存取控管：資料庫存取透過 RLS 策略控管，確保使用者僅能存取自己的資料
  • 安全稽核：定期進行安全稽核與弱點掃描
  • 人員管控：僅經授權之人員得在業務必要時存取個人資料
  • 事件應變：備有資料外洩事件應變計畫，確保於法定時限內通知受影響之使用者

3.3 資料保存期限
