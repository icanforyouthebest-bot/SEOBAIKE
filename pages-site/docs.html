API 端點

以下列出所有可用的 API 端點，包含約束推理引擎、AI 閘道、產業節點查詢與路徑驗證

POST /functions/v1/nvidia-boss 核心引擎
約束推理引擎
SEOBAIKE 主要 AI 推理引擎。接收使用者提示詞後，經過 L1-L4 約束層驗證，產生符合專利規範的推理結果。支援多輪對話與上下文記憶。
POST /functions/v1/ai-gateway 閘道路由
AI 閘道
多平台 AI 路由閘道。根據請求內容自動分配至最適合的 AI 夥伴（OpenAI、Anthropic、DeepSeek 等），實現負載均衡與容錯切換。
GET /rest/v1/l1_categories 約束結構
L1 產業類別查詢
查詢 L1 層級的宏觀產業分類。包含 TSIC、NAICS、NACE、JSIC 多國標準代碼對齊。支援 is_frozen 欄位篩選及分頁查詢。
GET /rest/v1/l2_subcategories 約束結構
L2 次產業查詢
查詢 L2 層級的次產業分類。透過外鍵關聯至 L1 類別，可使用 select 參數展開父層資料。支援 PostgREST 過濾語法。
GET /rest/v1/l3_processes 約束結構
L3 製程查詢
查詢 L3 層級的製程與作業類型。透過外鍵關聯至 L2 次產業，涵蓋各產業的標準化製程定義與工序描述。
GET /rest/v1/l4_nodes 約束結構
L4 原子節點查詢
查詢 L4 層級的原子級工業節點。每個節點為約束結構中最細粒度的單元，包含 766 個產業節點的完整定義與屬性。
POST /rest/v1/rpc/check_inference_path 路徑驗證
路徑驗證
驗證 L1 → L2 → L3 → L4 推理路徑是否合法。每次 AI 動作前必須先呼叫此端點。回傳 allowed / denied / halted 狀態。這是專利 TW-115100981 的核心機制。

程式碼範例

使用 curl 呼叫 API 的完整範例，可直接複製貼上至終端機執行

呼叫約束推理引擎

bash 
# 呼叫 SEOBAIKE 約束推理引擎
curl -X POST \
  https://vmyrivxxibqydccurxug.supabase.co/functions/v1/nvidia-boss \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "分析半導體產業的 L1-L4 約束路徑",
    "max_tokens": 1024,
    "l1_id": "manufacturing",
    "l2_id": "semiconductor"
  }'

驗證推理路徑

bash 
# 驗證 L1 → L2 → L3 → L4 路徑是否合法
curl -X POST \
  https://vmyrivxxibqydccurxug.supabase.co/rest/v1/rpc/check_inference_path \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "apikey: YOUR_ANON_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "p_l1_id": 1,
    "p_l2_id": 5,
    "p_l3_id": 12,
    "p_l4_id": 42
  }'

查詢 L1 產業類別

bash 
# 取得所有 L1 產業類別
curl -X GET \
  "https://vmyrivxxibqydccurxug.supabase.co/rest/v1/l1_categories?select=*" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "apikey: YOUR_ANON_KEY"

# 查詢特定類別並展開 L2 子分類
curl -X GET \
  "https://vmyrivxxibqydccurxug.supabase.co/rest/v1/l1_categories?select=*,l2_subcategories(*)&id=eq.1" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "apikey: YOUR_ANON_KEY"

呼叫 AI 閘道

bash 
# 透過 AI 閘道路由至最佳 AI 夥伴
curl -X POST \
  https://vmyrivxxibqydccurxug.supabase.co/functions/v1/ai-gateway \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "messages": [
      {"role": "user", "content": "分析台灣製造業趨勢"}
    ],

    "max_tokens": 2048
  }'

速率限制

為確保服務品質與公平使用，各端點設有不同的速率限制

端點 限制 時間窗口 備註
約束推理引擎 60 次 每分鐘 含 L1-L4 路徑驗證
AI 閘道 120 次 每分鐘 依 AI 夥伴各自限制
L1-L4 查詢 300 次 每分鐘 RESTful 讀取操作
路徑驗證 200 次 每分鐘 RPC 呼叫
全域限制 1,000 次 每小時 所有端點合計

超過速率限制將收到 HTTP 429 回應，請等待 retry_after 秒後再重試。需要更高的配額請聯繫 seobaike@aiforseo.vip。
