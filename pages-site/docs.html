入門

SEOBAIKE 是什麼？

SEOBAIKE 是由小路光有限公司打造的 AI 約束即服務（CaaS）平台。我們的定位是「AI 界的 App Store」——不製造 AI 模型，而是打造讓 AI 安全運行的容器。
  • 透過 L1-L4 四層約束架構控制 AI 推理路徑
  • 插拔式設計，支援多種 AI 模型自由接入
  • 所有 AI 服務透過 aiforseo.vip 通路統一管理

如何註冊 SEOBAIKE？

註冊流程：
  • 前往 登入頁面，選擇「建立帳號」
  • 填寫電子郵件與密碼
  • 驗證信箱後即可開始使用
註冊完全免費，瀏覽平台不收費。

如何在平台上架商品或服務？

上架流程：
  • 登入後進入 市集頁面，點選「我要上架」
  • 填寫商品名稱、說明、定價等基本資訊
  • 提交後經平台審核，通過即上架
不需要寫程式、不需要架設伺服器。支援數位商品、AI 工具、設計素材、教學課程、顧問服務等。

SEO Points

什麼是 SEO Points？

SEO Points 是 SEOBAIKE 平台的通用點數，作為平台內所有交易與服務的計價單位。
  • 購買市集商品與 AI 服務
  • 支付 API 呼叫費用
  • 解鎖進階功能

如何購買 SEO Points？

購買方式：
  • 登入帳號後，前往 方案頁面 選擇適合的方案
  • 支援信用卡及多種國際付款方式
  • 購買後點數即時入帳，可在儀表板查看餘額

SEO Points 怎麼使用？

SEO Points 用途涵蓋平台所有服務：
  • 市集購買 — 購買 AI 工具、模板、素材等商品
  • API 呼叫 — 依使用量自動扣點
  • 進階功能 — 解鎖高階數據分析、批量操作等
所有消耗明細皆可在儀表板即時查看。

賣家

如何成為賣家？

成為賣家的流程：
  • 註冊並登入 SEOBAIKE 帳號
  • 進入賣家中心，填寫賣家基本資料
  • 提交申請，平台審核通過後即可開始上架
審核重點包含商品品質、服務描述的準確性，確保買家獲得良好體驗。

分潤制度如何運作？

分潤依合約約定。所有交易紀錄透明可查，即時顯示在賣家儀表板。具體分潤比例請參閱您的賣家合約，或聯繫我們取得詳細說明。

商品審核標準是什麼？

平台審核關注以下面向：
  • 內容品質 — 商品描述清楚、功能符合宣稱
  • 合規性 — 不得包含違法、侵權或有害內容
  • 技術規範 — API 服務需符合平台技術標準
  • 使用者體驗 — 介面友善、文件完整
審核結果會透過電子郵件通知。未通過時會附上具體改善建議。

買家

如何購買市集商品？

購買流程：
  • 瀏覽 市集，找到想要的商品
  • 查看商品詳情與評價
  • 使用 SEO Points 完成購買
  • 購買後可在儀表板管理已購商品
平台 AI 也會根據您的需求提供商品推薦。

退款政策是什麼？

SEOBAIKE 重視買家權益：
  • 數位商品依平台退款條款處理
  • 如商品與描述嚴重不符，可提出爭議申請
  • 爭議由平台客服團隊審核處理
詳細退款條款請參閱 服務條款。如有任何問題，歡迎 聯繫我們。

技術

SEOBAIKE 提供 API 嗎？

是的，SEOBAIKE 提供完整的 RESTful API：
  • 約束驗證 API — 呼叫 check_inference_path() 驗證 L1-L4 推理路徑
  • 市集 API — 商品查詢、購買、管理
  • 數據 API — 銷售分析、使用統計
API 文件持續更新中。開發者可透過 SEO Points 支付 API 使用費用。

平台使用什麼 AI 引擎？

SEOBAIKE 採用插拔式架構，不綁定單一 AI 模型：
  • 支援多種大型語言模型接入
  • 所有模型必須通過 L1-L4 約束驗證才能執行推理
  • 企業可根據需求自由選擇最適合的 AI 引擎
核心理念是：我們不製造 AI，我們讓 AI 安全運行。

平台資料安全嗎？

安全是 SEOBAIKE 的最高優先。平台採用：
  • 全程加密傳輸 — 所有資料傳輸經 TLS 加密
  • 嚴格存取控制 — 角色分層權限管理（RLS）
  • 符合法規 — 遵循台灣個資法規定
  • 定期稽核 — 持續性安全檢測與漏洞修補
詳細資訊請參閱 隱私權政策。
