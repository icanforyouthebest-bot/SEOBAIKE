常見問題

關於定價與方案的疑問，我們在這裡解答

免費方案每月提供 1,000 次 API 呼叫與 100 次約束驗證，支援 1 個 AI 模型路由及基礎 Dashboard。適合個人開發者體驗平台功能、小型專案測試，或是評估 SEOBAIKE 是否符合需求的企業。免費方案不包含 Webhook、API Key 管理及白標部署功能。

API 呼叫次數是您透過 SEOBAIKE 平台發出的所有 API 請求總數，包含查詢、資料寫入、模型路由等。約束驗證次數則是呼叫 check_inference_path() 進行 L1→L2→L3→L4 路徑驗證的次數。一次 API 呼叫可能觸發零次或多次約束驗證，取決於業務邏輯的設定。兩者獨立計算，皆可在 Dashboard 即時查看。

不會直接斷線。當您接近方案上限的 80% 時，系統會透過 Dashboard 和 Email 發送提醒。達到上限後，免費方案會暫停服務至下個月重置；專業方案則提供 10% 的彈性超額空間，超出部分按比例計費（每 1,000 次呼叫 NT$0.08）。企業方案為無限額度，不受此限制。

所有方案皆無綁約限制，您可以隨時在帳戶設定中升級或降級。升級立即生效，依剩餘天數比例計費；降級則在當期結束後生效。您的約束規則設定和稽核紀錄不會因方案變更而遺失，但降級後超出方案範圍的功能將暫停存取。年繳方案提供 8 折優惠，中途取消可依比例退款。

白標部署讓您可以用自己的品牌和網域提供 SEOBAIKE 平台服務，適合將 AI 約束推理整合進現有產品的 SaaS 企業。自訂約束規則允許您定義超越標準 L1-L4 架構的企業專屬規則，例如特定產業法規、公司內部合規政策等。這些功能由專屬客戶經理協助配置，並享有 99.99% SLA 保證。
